P2Peek

Project Description

P2Peek is a decentralized communication tool that provides secure, peer-to-peer (P2P) connections for file transfers and messaging. Built using Python and TCP networking, it allows users to establish direct, encrypted connections, bypassing centralized servers and reducing vulnerabilities associated with data breaches or third-party monitoring.

With a focus on simplicity and privacy, P2Peek is designed to be accessible to both technical and non-technical users, offering a reliable alternative to traditional communication platforms. As concerns over data privacy grow, P2Peek meets the increasing demand for secure, private communication, making it ideal for personal use and enterprise settings alike.

The story

Existing communication models rely on centralized servers, making them vulnerable to data breaches, surveillance, and server outages. This centralized dependency risks user privacy and limits control over data security. P2Peek addresses these issues by enabling direct, peer-to-peer connections, ensuring secure, private communication without reliance on third-party servers.

Project Description

ChroMakit is a web-based application that allows users to apply real-time color filters and color swapping to live video streams captured by their camera. Using advanced computer vision techniques, ChroMakit enables users to isolate specific colors within the video and dynamically swap them with different colors.

Built with modern web technologies, the application provides an engaging, interactive experience that can be utilized for creative, educational, or practical purposes. Its straightforward interface makes it accessible to users of all skill levels, turning everyday video streams into customizable, visually captivating content in real-time.

The story

In traditional video communication, color-based customization and real-time visual effects are limited, restricting creative and practical uses. Existing platforms lack flexible, user-controlled color isolation and swapping features, resulting in static visual experiences. ChroMakit addresses this by enabling dynamic color adjustments, enhancing interactivity and personalization in live video streams.

OUR APPROACH

ChroMakit utilizes real-time computer vision to isolate and replace colors in live video streams, creating customizable visual effects. By capturing the video feed from the user’s camera, the app applies advanced filtering algorithms, enabling dynamic color adjustments. This approach enhances interactive experiences, making video communication more engaging and personalized.

Project Description

The IP Legitimacy Checker is a security-focused tool designed to verify the authenticity and trustworthiness of IP addresses in network environments. By analyzing IP information, it helps users distinguish legitimate from suspicious or potentially malicious IP addresses.

This tool plays a crucial role in network security, providing organizations with an efficient way to monitor and filter out risky connections. Its functionality may include checking IP addresses against known blacklists, geographical data, and threat databases, making it valuable for preventing unauthorized access, phishing, and DDoS attacks. The IP Legitimacy Checker enhances overall network safety, safeguarding sensitive data and preventing security breaches.

The story

A close friend of mine launched an online business but faced devastating DDoS attacks from suspicious IPs, leading to downtime and frustrated customers. As their business struggled, they turned to me for help. Inspired by their plight, I created the IP Legitimacy Checker to help users identify and block malicious IPs, safeguarding their online spaces and preventing similar attacks, ensuring that others wouldn't experience the same challenges.

OUR APPROACH

The IP Legitimacy Checker employs advanced algorithms to analyze and verify IP addresses against known databases and blacklists. By assessing various attributes, it identifies potentially malicious connections, allowing users to filter out threats. This proactive approach enhances network security and empowers users to protect their online environments effectively.
